Create a production-ready portfolio website for a Senior UI/UX Designer, Graphic Designer, Content Creator, Team Lead, and Video Editor.

The website should not feel like a traditional portfolio.

It should feel like an award-winning interactive digital experience.

Do not create a project grid.

Do not create a typical About, Services, Portfolio, Testimonials structure.

The portfolio should revolve around only 3 flagship case studies.

These 3 projects are the entire experience.

====================================================
CORE CONCEPT
============

The visitor enters a cinematic storytelling experience.

Instead of browsing projects, they journey through them.

The website should feel like:

* Awwwards Site of the Day
* Interactive editorial website
* Digital museum exhibition
* Modern luxury product launch experience

====================================================
OPENING EXPERIENCE
==================

Create a cinematic intro.

The visitor lands on a full-screen geometric mask.

Inside the mask is a zoomed-in image.

The user cannot immediately understand what they are looking at.

As they begin scrolling:

* The mask expands.
* The image zooms out.
* The project title slowly appears.
* Layers move independently.
* The image gains depth.

The reveal should feel dramatic and intentional.

SCROLL STORYTELLING
===================

The entire website is driven by scroll.

As the user scrolls:

* Images reveal
* Headlines animate
* Elements move in depth
* Content unfolds naturally

Motion should feel cinematic.

Never flashy.

Never distracting.

====================================================
PARALLAX SYSTEM
===============

Create depth using multiple layers.

Background Layer:
Moves slowly.

Middle Layer:
Moves normally.

Foreground Layer:
Moves slightly faster.

Device mockups:
Float subtly.

Images:
Shift in perspective.

Create a premium sense of space.
